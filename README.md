# md-toc

Table of Contents generator for Markdown.


# Installation

```sh
$ git clone https://github.com/pocke/md-toc
$ cd md-toc
$ make
```

# Usage

## Basic Usage

```sh
$ md-toc < README.md
-  md-toc
-  Installation
-  Usage
  -  Basic Usage
  -  Options
-  TODO
```
